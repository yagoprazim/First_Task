Resumo do que eu aprendi no 1º Dia de "Workshop"

Aprendi um pouco sobre as chamadas:"Linguagens de Marcação" (HTML + CSS). Foi me ensinado que, enquanto o HTML é utilizado para estruturar um Site, o CSS já serve para estilizar um Site, deixando-o mais bonito e mais agrádavel para os usuários.

Assimilei diversas Tags e Atributos que eu ainda não conhecia, pois, o pouco que eu sabia de tais assuntos, aprendi nas poucas semanas de aulas que tive até então (é tudo muito novo para mim). Percebi que os Atributos possuem uma função de modificar as propriedades básicas de uma Tag, deixando-as mais organizadas de acordo com a necessidade do desenvolvedor.

Aprendi também a usar melhor as ferramentas Git e GitHub. Entendi que, o Git é um programa que serve para que o desenvolvedor possa controlar o versionamento dos seus códigos de forma organizada, gerando, inclusive, um repositório local. Em seguida, entra o papel do GitHub, o qual tem a função de fazer um upload do nosso repositório Local para o Remoto, permitindo que seja possível o acesso de outros desenvolvedores ao nosso repositório público.

Muitos detalhes foram expostos acerca das Tags e Atributos de HTML e CSS, mas o que entendi é que, apesar de não ser possível decorar todas elas de uma hora para outra, o importante é eu ir entendendo aos poucos as funções que elas exercem no desenvolvimento de um código.
